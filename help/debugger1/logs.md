# Logboeken{#logs}

Het Logs-scherm biedt specifieke informatie over DTM-implementaties. Het gereedschap Foutopsporing console voor DTM hoeft niet te zijn ingeschakeld om deze informatie weer te geven. U kunt filteren op oplossingen die zijn ge√Ømplementeerd via de bijbehorende DTM-gereedschappen. U kunt kiezen welke van de volgende prioriteitsniveaus op het scherm worden weergegeven:

* Logboek
* Info
* Waarschuwingen
* Fouten

![](assets/logs.jpg)

Sommige DTM-logitems bevatten een optie Code tonen in het bericht. Klik **[!UICONTROL Show Code]** om de voorwaardelijke code te bekijken die bepaalt of een regel zou moeten in brand steken.
